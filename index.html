第1章 导论

金融工程学是将金融学、数学、统计学、计算机科学结合的交叉学科，核心目标是通过创新工具设计、资产定价和风险管理来解决复杂金融问题。

衍生品市场的核心分类

按交易场所

  • 交易所交易：标准化合约，中央清算，无违约风险
  • 场外交易（OTC）：个性化定制，有信用风险

按合约性质

  • 远期承诺（Forward Commitments）：远期、期货、互换——双方都有义务履行
  • 或有索取权（Contingent Claims）：期权——买方有权利无义务

金融工程的主要分析方法

  • 无套利定价法：相同现金流的两项资产价格必须相等
  • 复制组合：用已知价格的资产组合复制目标资产的现金流
一价定律：在竞争市场中等值资产价格应相等。套利：违反一价定律时的无风险获利机会。
理解远期承诺vs或有索取权的区别。真题常出现合成(synthetic)构造题目。

第2章 远期和期货定价

复利基础

每年m次复利终值：A(1+R/m)mn
连续复利终值：AeRcn
转换：Rc=m ln(1+R/m)

远期定价公式

不支付收益的资产

F = S erT

例：S=$40, r=5%, T=3月 → F=40×e0.05×3/12=$40.50

支付已知现金收益

F = (S-I) erT

支付已知红利率q

F = S e(r-q)T
核心思路：构造无风险组合（借入S买资产+卖远期），无套利迫使F等于持有成本。

远期合约价值

多头价值 f = (F-K) e-rT
不支付收益：f = S - K e-rT
区分远期价格F(新合约交割价)与合约价值f(已有合约盈亏)。

第3章 期货的对冲策略

基本对冲策略

空头套期保值

  • 持有资产或将在未来出售→卖出期货，锁定卖出价

多头套期保值

  • 将在未来购买→买入期货，锁定买入价

基差风险

基差 b = 现货S - 期货F

空头对冲有效价：S2+(F1-F2)=F1+b2

空头：基差扩大有利。多头：基差缩小有利。

合约选择原则

  1. 选择相关性最高标的
  2. 选交割月在到期之后的最近月份
  3. 避免交割月持有

Beta对冲

N=β×S/F
调整Beta：N=(βT-β*)×S/F
高频题型：对冲合约数计算、基差分析、Beta对冲。

第4章 利率期货

美国国债报价

现金价=报价+应计利息；报价以1/32为单位

长期国债期货

  • 标的：面值$100,000，息票率8%标准债券
  • 空头方选择最便宜可交割债券(CTD)
空头方收到现金=期货报价×转换因子+应计利息

Eurodollar期货

期货价=100-R；1基点=$25/合约

远期利率协议(FRA)

  • 多头(借款方)：支付固定RK，收到LIBOR
  • 空头(贷款方)：收到固定RK，支付LIBOR

久期

麦考利久期D=[Σt×PV(CFt)]/债券价
修正久期=D/(1+y)
ΔP/P≈-D×Δy

基于久期的对冲

N = DB×S / (DF×F)
常见题型：久期计算、债券期货对冲合约数、CTD判断。

第5章 互换

利率互换基本概念

  • 交换固定↔浮动利率现金流
  • 名义本金不交换

比较优势理论

  • 不同信用等级借款人在不同市场有比较优势

互换估值：债券法

互换价值=V浮动-V固定
V固定=Σ(固定利息×e-riti)+本金×e-rntn

FRA组合法

每期浮动现金流看作一份FRA，加总各期价值

互换利差

互换利差=互换利率-同期限国债收益率
题型：互换估值、互换利率计算。参考习题2025-1第2题。

第6章 股票期权价格的特性

看涨内在价值=max(S-X,0)
看跌内在价值=max(X-S,0)

影响因素

变量看涨看跌
S标的资产↑↑↓
X执行价↑↓↑
σ波动率↑↑↑
r利率↑↑↓
D红利↑↓↑

价格上下界

欧式看涨：c≥max(S-Xe-rT,0)，c≤S
欧式看跌：p≥max(Xe-rT-S,0)，p≤Xe-rT

看跌-看涨平价

欧式(无红利)：c+Xe-rT = p+S
美式：S-X≤C-P≤S-Xe-rT

美式提前执行

  • 看涨(无红利)：不提前
  • 看跌(无红利)：可能提前
  • 看涨(有红利)：除息日前可能提前
必考：看跌-看涨平价。参考习题2025-2第6题。

第7章 二叉树模型

单步二叉树

p = (erΔt-d)/(u-d)
f = e-rΔt[p·fu+(1-p)·fd]
Δ = (fu-fd)/(Su-Sd)

两步二叉树

  • 非重迭树：2n结果
  • 重迭树：n+1节点

美式期权处理

每节点比较：持有价值vs立即执行价值，取较大者。

必考题：两步二叉树。参考习题2025-1第4题。

第8章 股票价格变动的模型

  • 马尔可夫过程：未来只取决于当前值
  • 维纳过程：dz=ε√dt，ε~N(0,1)
广义维纳过程：dx=adt+bdz
几何布朗运动：dS/S=μdt+σdz

伊藤引理

dG=(∂G/∂S·μS+∂G/∂t+½∂²G/∂S²·σ²S²)dt+∂G/∂S·σSdz

G=lnS：d(lnS)=(μ-σ²/2)dt+σdz

lnST~N(lnS0+(μ-σ²/2)T,σ²T)

第9章 股票期权定价公式(B-S)

B-S-M微分方程

∂f/∂t+rS·∂f/∂S+½σ²S²·∂²f/∂S²=r f
假设：股价连续、波动率常数、无风险利率常数、无交易成本、无红利、无套利、可连续对冲。

Black-Scholes公式

c=SN(d1)-Xe-rTN(d2)
p=Xe-rTN(-d2)-SN(-d1)
d1=[ln(S/X)+(r+σ²/2)T]/(σ√T)
d2=d1-σ√T

含红利率q

c=Se-qTN(d1)-Xe-rTN(d2)

隐含波动率

  • 市场价格代入B-S反推出的σ
  • 波动率微笑：不同执行价隐含波动率不同
  • VIX指数：S&P500期权30天隐含波动率
必考：B-S公式、已知Delta反推S。参考习题2025-2第1题。

第10章 希腊字符与期权对冲

五大希腊字母

字母看涨公式含义
ΔN(d1)S变$1期权变多少，涨0~1跌-1~0
ΓN'(d1)/(Sσ√T)Delta对S敏感度，>0
Θ-SN'(d1)σ/(2√T)-rXe-rTN(d2)时间衰减，通常负
ν(Vega)SN'(d1)√T波动率变化敏感度
ρ(Rho)XTe-rTN(d2)利率敏感度

Delta对冲

每卖1份看涨，买入Δ份股票
看跌Delta=N(d1)-1
Delta对冲需动态调整。仅Delta无法对冲Gamma风险。

Delta-Gamma对冲

  1. 先选另一期权使Gamma中性
  2. 再调标的资产使Delta中性
参考习题2025-2第5、6题(D-G对冲、动态Delta调整)。

📋 真题与习题精选

以下是从习题和真题中提取的核心题型。考前务必动手计算。

题型1：远期/期货定价与套利

例：S=$40, r=5%, 3月远期F=40×e0.05×3/12=$40.50。若F=$43则套利获利$2.50。

题型2：期货对冲

  • Beta对冲：N=β×S/F
  • 久期对冲：N=DBS/(DFF)

题型3：看跌看涨平价

例：S=16000, X=16000, c=677.146, r=2%, T=3月
p = 677.146+16000e-0.02×0.25-16000

题型4：B-S公式

例：c=0.74, Δ=0.3426, N(d2)=0.31728, X=50, r=4%, T=3月
c=SN(d1)-Xe-rTN(d2)→ S=(0.74+50×0.99005×0.31728)/0.3426

考前必记公式

1.F=SerT
2.c+Xe-rT=p+S
3.f=e-rΔt[pfu+(1-p)fd]
4.c=SN(d1)-Xe-rTN(d2)
5.d1=[ln(S/X)+(r+σ²/2)T]/(σ√T)
6.Δ=N(d1)
7.N=β×S/F
8.N=DBS/(DFF)
易错点：远期价格vs合约价值；连续vs年复利转换；欧式/美式二叉树差异；Delta需动态调整；看跌Delta=N(d1)-1
